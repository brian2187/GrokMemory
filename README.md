# GrokMemory

**Persistent Memory & Work Area for Grok + Brian Moore**

- Version-controlled backup and expansion of GrokM email system.
- Primary: Grok email folder (GrokM items).
- Secondary: This GitHub repo for history, diffs, collaboration, and advanced features.
- Connectors: GitHub (full write), Outlook (Calendar + Mail).

## Structure
- `memory/` - Master Profile, snapshots
- `directives/` - Primary/Secondary Directives, Operating Principles
- `logs/` - Session logs, action plans
- `experiments/` - AI tests, imitation experiments, new features
- `skills/` - Custom skills and tools
- `.github/workflows/` - Automation (future syncs, checks)

## Capabilities Leveraged
- Full Git version control & history
- Branching for experiments
- Issues/PRs for task tracking
- Actions for potential automation
- Collaborators for multi-user if needed

Updated via Grok on refreshes and user sessions. Email folder as reliable backup.
